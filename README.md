# FinERP — Современная ERP-система для бизнеса Казахстана

Полноценная замена 1С: бухгалтерия, склад, касса, банк, кадры, документы.
Все расчёты по **Новому Налоговому Кодексу РК 2026** (ЗРК 214-VIII от 18.07.2025).

## 🚀 Быстрый старт

### 1. Клонировать репозиторий
```bash
git clone https://github.com/YOUR_USERNAME/finerp-kz.git
cd finerp-kz
npm install
```

### 2. Создать проект Supabase
1. Зайдите на [supabase.com](https://supabase.com) и создайте новый проект
2. Откройте **SQL Editor** и выполните содержимое файла `supabase/schema.sql`
3. В **Authentication > Settings** включите Email provider
4. Скопируйте URL и anon key из **Settings > API**

### 3. Настроить переменные окружения
```bash
cp .env.local.example .env.local
```
Заполните:
```
NEXT_PUBLIC_SUPABASE_URL=https://YOUR_PROJECT.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_ANON_KEY
ANTHROPIC_API_KEY=YOUR_KEY  # опционально, для AI Жанара
```

### 4. Запустить
```bash
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000)

### 5. Деплой на Netlify
1. Подключите GitHub репозиторий к Netlify
2. Build command: `npm run build`
3. Publish directory: `.next`
4. Добавьте переменные окружения в Netlify

| Модуль | Описание |
|--------|----------|
| **Документы** | Счета, накладные, акты, ПКО/РКО, платёжные поручения, договоры, ТТН, доверенности — 12 типов |

| **Банк** | Банковские операции, платёжные поручения |
| **Кадры и ЗП** | Сотрудники, расчёт ЗП по НК РК 2026 |
| **Отчёты** | ФНО 910, 200, 300, 100, ОСВ, баланс |
| **НК РК 2026** | Справочник всех ставок и изменений |
| **AI Жанара** | AI-бухгалтер (Anthropic Claude) |
| **Настройки** | Реквизиты организации, контрагенты |
